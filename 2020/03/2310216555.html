python程序打包成exe文件
发表于 2020-03-31| 更新于 2020-07-03|Experience
字数总计:306|阅读时长: 1 分钟
|阅读量:|评论数:

  • step1：安装pyinstaller，在cmd窗口使用pip install pyinstaller安装

  • step2：cd 到你的文件目录cd D:\Pycharm Destination\ (test.py文件所在的目录)

  • step3：运行pyinstaller -F test.py

  • step4：看结果，结果在新生成的文件夹dist下就有test.exe

遇到的错误：

  1. 运行完“pyinstaller -F test.py”命令后，提示缺少一堆api文件。

    解决办法：将C:\Windows\System32\downlevel目录下的所有文件复制到“F:\python\DLLs”目录下

  2. 上面的问题解决后，又报出ModuleNotFoundError: No module named 'setuptools._vendor'的错误

    解决办法：使用pip install --upgrade setuptools 更新到最新版本后问题解决

 

补充:

pyinstaller -F -i test.ico -w test.py

-w为在GUI程序中略去控制台的显示效果，test.ico为想要的图标，需要和test.py在同一目录下

 

 

注：程序中如果有中文，则会报错

 解决办法：不要写完程序后复制粘贴到一个txt文本，再转成py文件。这样会不支持中文。正确的做法是：在	   编辑器里写完程序直接保存，然后去找存储路径下的py文件，那个文件里有中文也没问题。

 

多个文件：

pyinstaller -F -i 111.ico -w mian.py -p utils.py

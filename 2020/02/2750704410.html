1. 安装fancybox功能

fancybox功能主要用于实现相册图片的点击放大。安装过程如下：

打开站点配置文件_config.yml，搜索fancybox字段，设置其值为true
进入到theme/next/文件夹下，打开git bash。执行如下命令：
git clone https://github.com/theme-next/theme-next-fancybox3 source/lib/fancybox
